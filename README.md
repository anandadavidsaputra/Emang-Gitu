# 🎬 Emang Gitu

Emang Gitu adalah website quiz interaktif yang dirancang untuk menentukan tipe atau gambaran kepribadian pengguna berdasarkan jawaban yang dipilih.

---

## 🌐 Live Demo

👉 [https://anandadavidsaputra.github.io/Emang-Gitu/](https://anandadavidsaputra.github.io/Emang-Gitu/)

---

## 🧩 Overview

Emang Gitu dibangun sebagai proyek latihan pengembangan web untuk:

* menampilkan pertanyaan quiz secara interaktif
* mengolah jawaban pengguna
* menghitung dan menentukan hasil kepribadian
* menampilkan hasil akhir berdasarkan skor atau pilihan terbanyak

Website ini berfokus pada implementasi logika perhitungan sederhana menggunakan JavaScript serta manipulasi DOM untuk menampilkan pertanyaan dan hasil secara dinamis.

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🎯 Purpose

Website ini dibuat untuk tujuan pembelajaran sekaligus menambah proyek dalam portofolio pengembangan frontend, khususnya dalam membangun sistem quiz berbasis logika JavaScript.
